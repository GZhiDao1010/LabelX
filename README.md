# LabelX
Koa2 游戏基础框架

> 小型游戏服务端开发框架，可以做二次开发

```├── app.js 
├── common  控制器/mysql持久化服务/socket.io服务
├── config  配置文件
├── coverage 代码覆盖率
├── logs  日志
├── middlewares 自定义中间件
├── modules  sequelize Model
├── public  静态文件托管目录
├── routes 路由集合
├── test  单元测试
├── utils 工具集合
├── package.json
├── package-lock.json
└── views  视图
```
